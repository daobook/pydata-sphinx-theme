从主题 sections 添加/删除项#

有几个主要的主题 section，你可以自定义添加/删除组件，或添加自己的组件。每个 section 都配置了一组 html templates —— 这些是由 Sphinx 插入到该 section 的 HTML 片段。

你可以选择哪些模板出现在每个 section，以及它们出现的顺序。这个页面描述了您可以自定义的主要领域。

备注

当在每个 section 配置模板时，如果你愿意，你可以在每个模板后面省略后缀 .html。

navbar 项#

导航条位于页面顶部，分为三个 section。每个 section 在 conf.py 中配置如下配置：

  • 左 section：html_theme_options['navbar_start']

  • 中间菜单：html_theme_options['navbar_center']

  • 右边 section：html_theme_options['navbar_end']

缺省情况下，采用如下配置：

html_theme_options = {
...
"navbar_start": ["navbar-logo"],
"navbar_center": ["navbar-nav"],
"navbar_end": ["navbar-icon-links"]
...
}

左侧边栏#

左边的侧边栏就在页面 main 内容的左边。它的配置与其他 section 的配置略有不同，因为配置侧边栏在 Sphinx 中是通过 html_sidebars 配置变量得到本机支持的。

对于左侧边栏，你可以配置模板，让它们只显示在特定的页面上。你可以在 conf.py 中这样配置：

html_sidebars = {
    "<page_pattern>": ["list", "of", "templates"]
}

任何匹配 <page_pattern> 的页面将插入各自的模板。你也可以用 * 来做 glob 风格的匹配，也可以用 ** 来匹配所有的页面。

缺省情况下，配置如下：

html_sidebars = {
    "**": ["search-field", "sidebar-nav-bs", "sidebar-ethical-ads"]
}

左侧边栏结束部分#

在页面底部的左边栏中有一个特殊的 <div>，而不管它上面的内容是什么。

要控制这个 div 中的 HTML 模板，请在 conf.py 中使用 html_theme_options['left_sidebar_end']。

默认情况下，它有以下模板：

html_theme_options = {
  ...
  "left_sidebar_end": ["sidebar-ethical-ads"],
  ...
}

右边的页内侧边栏#

页面内的侧边栏就在页面 main 内容的右侧，在 conf.py 中配置了 html_theme_options['page_sidebar_items']。

默认情况下，它有以下模板：

html_theme_options = {
  ...
  "page_sidebar_items": ["page-toc", "edit-this-page"],
  ...
}

页脚#

页脚就在页面的 main 内容下面，并在 conf.py 中配置了 html_theme_options['footer_items']。

默认情况下，它有以下模板：

html_theme_options = {
  ...
  "footer_items": ["copyright", "sphinx-version"],
  ...
}

可以插入到 section 中的内置模板列表#

Below is a list of built-in templates that you can insert into any section. Note that some of them may have CSS rules that assume a specific section (and will be named accordingly).

  • icon-links.html

  • search-field.html

  • copyright.html

  • edit-this-page.html

  • last-updated.html

  • navbar-icon-links.html

  • navbar-logo.html

  • navbar-nav.html

  • page-toc.html

  • sidebar-ethical-ads.html

  • sidebar-nav-bs.html

  • sphinx-version.html

  • version-switcher.html

  • theme-switcher.html

将您自己的HTML模板添加到主题 section#

如果你想添加你自己的自定义模板到这些部分的任何一部分，你可以这样做以下步骤：

  1. 在名为 _templates 的文件夹中创建 HTML 文件。例如，如果你想使用 Jinja 模板显示文档的版本，你可以创建文件 _templates/version.html，并在其中放入以下内容：

    <!-- This will display the version of the docs -->
{{ version }}

  2. 现在，将该文件添加到您的菜单项上的上述部分之一。例如：

    html_theme_options = {
...
"navbar_start": ["navbar-logo", "version"],
...
}
